# Глоссарий

Единый словарь терминов. В **коде** (типы/идентификаторы/схемы) — эти слова. В **UI-копирайте** —
английский бренд Crown (ADR 0022): это только ярлыки поверх того же движка, механика не меняется.

**Бренд Crown ↔ движок** (UI-слово → термин кода):

| UI (Crown, английский) | Движок / код | Где |
|---|---|---|
| **Crown** (to crown) | donation / донат | донат зрителя стримеру |
| **Reign** | reputation / points / очки | локальная репутация |
| **rank**: Squire · Knight · Baron · Duke · King | tier / тир | лестница статуса (`src/lib/crown.ts`) |
| **The Crown** 👑 | сезонный топ-1 | титул, не порог |
| **realm** | channel / канал | пространство стримера |
| **Personal Space** (`/space`) | studio / студия владельца | Dashboard/Customization/Settings владельца |
| **Admin** (`/admin`) | ops / оператор | платформенная консоль оператора |

| Термин | Значение |
|--------|----------|
| **Локальная репутация** | Вычисляемое число статуса донора **внутри одного канала**. Глобального рейтинга нет. Непередаваема, непродаваема. |
| **Standing** | Кодовое имя движка. В UI (бренд Crown, ADR 0022) локальная репутация зовётся **Reign**, статус-лестница — **ранги** (Squire…King). |
| **Очки (points)** | Целочисленная единица репутации. Фиксировано: `points(d) = round(amount_usdc)`, 1 USDC = 1 очко (ADR 0007). |
| **Тир (tier)** | Уровень статуса по порогу очков. Настраивается стримером (имена/пороги — данные). В UI Crown на личных экранах статус показывается как **ранг** (Squire→King) по очкам Reign, независимо от названий тиров (`src/lib/crown.ts`). |
| **Бейдж (badge)** | Визуальный знак тира рядом с ником/сообщением/в лидерборде. |
| **Перк (perk)** | Что разблокирует тир. В ядре — флаг/описание (доступ, эмодзи, приоритет алерта). |
| **Донат** | Перевод USDC донор → стример (минус комиссия). В ядре **финален**. |
| **Донат-с-текстом** | Донат с сообщением. Только на активированном канале. Текст по умолчанию HELD. |
| **Комиссия** | ~3% от доната в трежери. Привязана к обороту, не к исходу. Расщепляет сам фронт. |
| **Трежери (treasury)** | Адрес оператора, куда идёт комиссия и сбор активации. Денег **доната** оператор не держит. |
| **Журнал репутации (ledger)** | Апенд-онли лог событий. Репутация = его детерминированная свёртка. |
| **Порог тира (threshold)** | Сколько очков нужно для тира/перков/участия в мини-играх. Единственный рычаг стримера в репутации (ADR 0007). |
| **HELD** | Текст пришёл, виден только стримеру и модераторам, ещё не опубликован. Дефолт. |
| **SHOWN** | Текст опубликован (оверлей / публичный фид). |
| **HIDDEN** | Текст не публикуется. Деньги и репутация донора остаются. |
| **QUARANTINED** | Текст изъят в закрытый бакет (нелегальщина / hard-block). |
| **Авто-показ** | Опц. режим публикации без ручного одобрения (с задержкой). Hard-block никогда не авто-показывается. |
| **Оверлей (overlay)** | Публичный вид для OBS: алерты, показанные сообщения, бейджи. |
| **Активация канала** | Одноразовый сбор (~$2 USDC) → `BASIC → ACTIVE`. Разблокирует донаты-с-текстом и публичные фичи. |
| **Канальный блок** | Стример блокирует кошелёк **на своём канале**. Не платформенно. |
| **Саспенд / бан** | Платформенные действия оператора: `SUSPENDED` (до ревью) / `BANNED` (подтверждено T&S). |
| **DISPUTE_LOST** | Протокольное списание репутации у инициатора проигранного ложного спора. Единственная отрицательная дельта — оператор репутацию не редактирует (ручного `ADMIN_VOID` больше нет, CR-1). Наказание нарушителя — блок кошелька/канала: обесценивает репутацию, не трогая честное число. |
| **Self-donation flood** | Главная атака: тролль донатит сам себе гадости в тексте. Сводится к платной саморазоблачающейся помехе. |
| **Sybil** | Один человек = много кошельков. Не закрываем (нет proof-of-personhood), облагаем фрикшеном. |
| **Некастодиальность** | Оператор не держит деньги доната и не имеет ключа над чужими средствами. |
| **DataProvider** | Интерфейс доступа к данным во фронте. Реализации: Mock → Api → Chain → Icp (см. `CLAUDE.md` §3). |
| **memo** | Поле атрибуции в ончейн-транзакции: `{creator_id, donation_id, msg_ref}`. |
| **ATA** | Associated Token Account — USDC-аккаунт адреса. ATA стримера создаётся в той же tx при отсутствии. |
| **Газовый флор** | Динамический минимум доната, растущий с ценой газа. |
| **T&S** | Trust & Safety: процесс и люди оператора по нелегальщине и абьюзу. |
| **CSE / CSAM** | Child Sexual Exploitation / Material. Обязательный карантин + preservation + репорт в NCMEC. |
